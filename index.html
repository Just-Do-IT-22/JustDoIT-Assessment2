Motivation

Currently, if you lose or damage an individual part beyond repair you have very limited options in replacing said part. In Japan, here is a domestic service where you can buy replacement part runners for your model kit through a mail in form included in every instruction manual. If you live outside of Japan however, the options to replace an individual piece mean that you either must buy a whole new model kit or use a replacement parts provider such as “Gundam Replacement Parts” for a Japan-based exporter or “Mecha Parts Guy” for a US-based exporter. Both these services work by buying extra model kits and selling either full runners, or individual pieces with the relevant numbers. This sees a problem as due to the nature of production; multiple pieces are cast at once onto plastic runners using injection moulded plastic so individual parts are currently impossible to purchase. Fans of Bandai’s Gundam models are already using the easily available resin 3D printers to create custom parts, so this would create a genuine way to create replacement parts that cuts down on plastic waste.

Description

My idea is to implement a partnership with Bandai in order to sell individually 3D resin printed pieces, made to order and selected through a website in which users select the piece/s they need replaced. The user would visit the website looking to replace a part that was broken or lost, search for the model kit that the piece is from by either entering the kit number or looking up the name of the kit and filtering relevant results. From there, the user would be greeted by dropdown menu’s that would show each runner name and after selecting the relevant runner, would show a following dropdown menu that then displays all the pieces which they can select the one they are searching for. Once the user was finished with their decision, they could continue to enter payment information and then shipping details and once processed, create an order for a replacement part.
The other side of this project would be the 3D printing process. A huge assumption of this project is that a partnership is formed with Bandai, as Gundam model kits are a trademarked entity and replicating these without permission could result in legal action from the owning parties. The CAD files that are used to design the individual parts would be shared converted into STL files to ready for 3D printing. When a user orders a particular replacement part, the relevant replacement part will be printed using the STL files from before on a small to mid-grade resin 3D printer. Multiple parts from different orders could be printed at once in order to reduce costs. After the parts are finished and cleaned up, they would be shipped to them via a courier or postal service, completing the order and helping the user finish their model project.

Tools and Technologies

For website development, services like Shopify could be used to create a storefront with built in payment handling support. These services also handle site creation and development using HTML and CSS. For the creation side of the project, software like AutoCAD could be used to view/modify existing CAD designs, and export to STL for printing. For 3D printers, there isn’t necessarily a need to use high end printers as the Gundam pieces are generally made from basic plastics (polystyrene, acrylic and ABS), so smaller hobby machines could be used originally and scaled up depending on the demand.

Skills Required

General website development knowledge would be important (HTML, CSS etc for front-end development and Ruby, Python, PHP etc to handle databasing of all the thousands of parts). Basic AutoCAD knowledge would be essential for handling the file conversions. But some design knowledge would be beneficial in case modifications needed to be made to the 3D models. Experience with 3D printers is another must-have although most entry-level and even some commercial grade 3D printers are quite beginner friendly, so this is less important than the above. Most of the skills above besides website development can be taught quickly to even the most beginning learners of this. For website development however, some moderate HTML and CSS knowledge would be required and the backend languages would need some more extensive knowledge in, especially for handling data in large volumes.

Outcome

If the project was successful, Gundam model builders all over wouldn’t have to worry about losing or destroying pieces and having to pay upwards of $450 for a brand-new model kit because they lost a single piece. When beginner model kits can begin at $20, it doesn’t seem like much of a problem to replace parts. However, with some of the high-end kits costing $450 and up it starts to become a huge problem for hobbyists. Another issue that this would help alleviate would be plastic reduction. As mentioned above in the production of plastic runners, to replace a single part would mean a whole new runner would have to be used to replace the part. This means the remaining plastic would go to waste as there are no non-Japanese Bandai plastic recycling options, which is a huge issue in modern society.
